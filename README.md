# Azure Finance Data Platform

End-to-end data engineering project built on Azure using
ADLS Gen2, Databricks, Delta Lake, Airflow, and Azure SQL.

## Architecture Overview
- Data Lake with Medallion Architecture (Bronze / Silver / Gold)
- Separation of code, compute, and storage
- Batch-oriented financial data ingestion via APIs
- Enterprise-style data governance, reliability, and optimization practices

## Tech Stack
- Azure Data Lake Storage Gen2
- Azure Databricks (Delta Lake)
- Apache Airflow
- Azure SQL Database
- GitHub + Jenkins (CI/CD)
- Jira (project tracking)
